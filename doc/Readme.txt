===============================================================================

　夏Conv

===============================================================================


■ソフトの概要

　動画・音楽ファイルの音量のノーマライズ（均一化）を行うソフトです。

　フォルダを指定して、その配下の動画・音楽ファイルを全て一括処理します。
　大量の動画・音楽ファイルを処理したい場合に便利ですよ。


　★当ソフトは ffmpeg を使用します。(外部コマンドとして呼び出します)
　　当ソフト使用前に ffmpeg を導入していただく必要があります。


■動作環境

　Windows 10
　確認していませんが Windows 7/8/8.1 でも動くかもしれません。

　.NET Framework 4 を使用します。


■インストール方法

　アーカイブの中身をローカルディスク上の任意の場所にコピーして下さい。


■アンインストール方法

　レジストリなどは一切使っていません。
　ファイルを削除するだけでアンインストールできます。


■ffmpeg導入方法

　1. ブラウザから https://ffmpeg.org/ を開いて下さい。

　2. ページ左側のメニューの Download をクリックして下さい。

　3. Get the packages の Windows のロゴにカーソルを合わせ、
　   Windows Packages の Windows Builds をクリックして下さい。

　4. Version の 3.2.4 を選択して下さい。

　　★バージョン番号が更新されている場合は Previous Builds からダウンロードして下さい。
　　　最新版で代替しても恐らく問題ありませんが、当ソフトが正常に動作しない可能性もあります。

　5. Architecture の Windows 64-bit または Windows 32-bit を「導入環境に合わせて」選択して下さい。
　   よくわからない場合は Windows 32-bit を選択して下さい。

　6. Linking の Shared を選択して下さい。(Static でも OK です)

　7. Download Build をクリックして .zip ファイルをダウンロードして下さい。

　8. ダウンロードした .zip を任意の場所に展開して下さい。
　   展開先のパスは後で使用するので、覚えておいて下さい。


　★この手順は 2017/3/13 に作成しました。(2019/7/8 修正)

　★開発・テスト環境では ffmpeg-3.2.4-win64-shared.zip を使用しています。

　★開発・テスト環境では C:\app\ffmpeg-3.2.4-win64-shared に展開しました。


■使い方

　1. natsuconv.exe を起動して下さい。

　2. 初回起動時は「お節介」ダイアログが表示されます。

　　2.1. 設定ダイアログを開きますか？というメッセージが表示されますので「はい」を選択して下さい。

　　2.2. ffmpeg のパスに ffmpeg を展開したパスを入力して下さい。
　　     右側の「...」ボタンを押せばフォルダ選択ダイアログから入力出来ます。

　　2.3. OK ボタンを押して下さい。

　3. 入力フォルダに、動画・音楽ファイルを格納している（大元の）フォルダを入力して下さい。
　   右側の「...」ボタンを押せばフォルダ選択ダイアログから入力出来ます。

　4. 開始ボタンを押して下さい。


　★サブフォルダも再帰的に検索され、入力フォルダの配下にある全ての動画・音楽ファイルが処理の対象になります。

　★この手順では、入力フォルダの配下にある動画・音楽ファイルが直接ノーマライズ（上書き）されます。
　　入力フォルダを変更せず、ノーマライズされた動画・音楽ファイルを別の場所へ出力したい場合は、
　　「入力ファイルと同じ場所に出力する」のチェックを外して下さい。

　★処理中は、なるべく入力・出力フォルダにアクセスしないで下さい。
